Documentation for `cremerlab-utils`
===================================

This site contains the documentation for `cremerlab-utils`, a Python package 
housing a bunch of useful software utilities used in the Jonas Cremer lab at 
Stanford University. It is still very much in alpha and should not yet be 
used beyond development.
